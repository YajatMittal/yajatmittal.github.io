# Yajat Mittal — Portfolio Website

Personal portfolio website showcasing my projects in AI, machine learning, computer vision, and software development.

## 🚀 About

This website highlights some of the projects I've built across AI/ML, computer vision, full-stack development, hackathons, and iOS development.

Built with a clean, minimal design using HTML, CSS, and React (loaded in-browser via CDN with Babel — no build step required).

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- React (via CDN + Babel standalone, JSX)

## 📁 Structure

```
.
├── index.html        # Entry point — loads React, Babel, and the app modules
├── styles.css        # All styling (themes, layout, background/spotlight effects)
├── assets/           # Icons, favicon, and images
└── src/
    ├── data.jsx       # Site content (about, experience, projects, skills)
    ├── icons.jsx      # Icon components
    ├── sections.jsx   # Section components (experience, projects, etc.)
    ├── app.jsx        # Root app + theme/layout controls
    ├── splash.js      # Intro splash screen
    └── tweaks-panel.jsx  # Live design-tweak controls
```
